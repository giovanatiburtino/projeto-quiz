[Link Demonstração](https://app-quiz-orcin.vercel.app/)

## 💿 Como rodar este projeto?

```bash

# Clone este repositório
$ git clone linkrepo

# Acesse a pasta do projeto no seu terminal
$ cd projeto-quiz

# Instale as dependências
$ npm install

# Execute a aplicação
$ npm run dev
```
